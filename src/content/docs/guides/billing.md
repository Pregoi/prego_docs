title: Billing
description: Plans, subscriptions, and payment integration in Prego.
---

Prego integrates billing so tenants can manage their plan and payment from the admin portal.

## Overview

- **Plans** — e.g. Free, Basic, Biz, Pro, Enterprise.
- **admin-web** — Billing page for viewing/updating plan and payment method.
- **Control-plane / Stripe** — Backend handles subscription state and payment provider integration.

## For tenants

Use the **Billing** section in the admin portal (m.pregoi.com) to view the current plan and payment details when enabled.
