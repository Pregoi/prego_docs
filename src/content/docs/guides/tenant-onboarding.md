Tenant onboarding covers the path from signup to a provisioned tenant.

## Flow overview

1. **Signup** — User signs up via admin-web (m.pregoi.com), often with subdomain/company details.
2. **Verification** — Email or other verification steps as configured.
3. **Provisioning** — Control-plane and/or workers provision the tenant.
4. **Access** — User can log in and use the admin dashboard, billing, and product.
