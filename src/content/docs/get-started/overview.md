Prego is a **SaaS platform** that provides tenant onboarding, billing, authentication, and product experiences.

## Main components

| Component | Purpose |
|-----------|--------|
| **www** | Marketing and product site (e.g. pregoi.com). |
| **admin-web (m.pregoi.com)** | Tenant admin: signup, login, dashboard, billing, profile, usage, users, portal. |
| **control-plane** | Backend APIs and services for tenant provisioning, billing, and configuration. |
| **Auth** | Authentication and session management. |
| **Billing** | Plans, subscriptions, and payment integration. |
