---
title: Glossary
description: Key terms used in Prego documentation.
---

| Term | Definition |
|------|------------|
| **admin-web** | The tenant-facing admin app (e.g. m.pregoi.com): signup, login, dashboard, billing, profile, usage, users, portal. |
| **Control Plane** | Backend services and APIs for tenant provisioning, configuration, and platform logic. |
| **Tenant** | An organization or workspace in Prego (e.g. one subdomain or billing account). |
| **JWT** | JSON Web Token used for authentication and API authorization. |
| **Prego** | The overall SaaS platform (www, admin-web, control-plane, auth, billing, etc.). |
