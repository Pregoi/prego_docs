## Where do I sign up or log in?

Use the **admin** app (e.g. m.pregoi.com). Signup and login pages are available there.

## How do I get API access?

Configure your app with the correct **Control Plane URL** and **Auth URL**. Use a valid JWT in the `Authorization` header when calling APIs. See [API overview](/api/overview/) and [Quick start](/get-started/quickstart/).

## Where are the full API specs?

OpenAPI (Swagger) specs are in the main Prego repo. This docs site gives an overview and links.

## How do I run Prego locally?

See [Quick start](/get-started/quickstart/) for running admin-web and calling the Control Plane API.
